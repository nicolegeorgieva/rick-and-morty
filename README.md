An app showcasing Rick & Morty characters, locations and episodes.

## Tech Stack

- **Languages:** `Kotlin`
- **Architectures:** `MVI`, `Clean`
- **Android:** [`Android Jetpack`](https://developer.android.com/jetpack), `Jetpack Compose`, `Hilt (DI)`, `Kotlin Flow`, `Kotlin Coroutines`, `KotlinX Serialization`, `Ktor`, `ArrowKt`, `Android SDK`
- **Other**: `HTTP`, `REST APIs`, `JSON`
